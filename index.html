FAQ

Preguntas frecuentes

¿Infrannova funciona en obras sin internet?
Sí. Modo offline integrado. Los datos se sincronizan cuando hay conexión disponible.
¿Se integra con SAP o sistemas ERP?
Sí. Integración nativa con SAP, Oracle y sistemas de gestión empresarial. API REST disponible.
¿Cuántas obras puede manejar?
Ilimitadas en planes Profesional y Enterprise. Básico hasta 3 obras simultáneas.
¿Hay app móvil?
Sí, disponible para iOS y Android. Optimizada para uso en terreno con modo offline.
¿Cómo es la implementación?
Onboarding guiado de 2 semanas con capacitación y soporte dedicado incluido.
¿Puedo migrar datos de otros sistemas?
Sí. Servicio de migración incluido con soporte técnico para una transición sin problemas.
¿Qué pasa con mis datos si cancelo?
Exportación completa de todos tus datos en formato estándar disponible en cualquier momento.
