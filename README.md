# infrannova / landing

Landing page de Infrannova — Gestión de Obras y Construction Management.

**Live:** https://infrannova-landing.vercel.app
**Repo:** https://github.com/Gigisanta/infrannova-landing

## Stack

- Single HTML file (standalone, no build step)
- Google Fonts: Plus Jakarta Sans + DM Sans + JetBrains Mono
- Lucide Icons via CDN
- CSS animations + IntersectionObserver

## Deploy

Push a `main` → deploy automático a Vercel.

## Estructura

```
infrannova/
├── landing/
│   ├── index.html
│   ├── .vercelignore
│   └── .git/           ← conectado a GitHub
└── README.md            ← (este archivo es del proyecto principal)
```
